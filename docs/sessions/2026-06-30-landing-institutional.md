## Что было сделано

### Страница `/ru/landing` — публичная главная

**Файлы:**
- `frontend/src/app/[locale]/landing/page.tsx` — Server Component, загружает Lexend + Source Sans 3
- `frontend/src/app/[locale]/landing/Screen.tsx` — Client Component, ~600 строк

**Секции страницы (по паттерну ui-ux-pro-max: Marketplace + Social Proof):**
1. **Nav** — sticky white, лого + 3 ссылки + Войти/Зарегистрироваться
2. **Hero** — navy фон `#0F172A` с grid-паттерном, H1 Lexend 52px, подзаголовок, dual CTA (Заказчик/Исполнитель), 3 trust-сигнала с зелёными иконками
3. **Stats-бар** — 94+ заявок / 230+ исполнителей / 12 городов / 98% сделок (Lexend 34px на blueDark)
4. **Как работает** — 4 пронумерованных шага с горизонтальной соединительной линией
5. **Для кого** — две карточки: Заказчики (белый фон, синие детали) / Исполнители (navy фон, белый текст)
6. **Преимущества** — 3 карточки: Верифицированные / Все регионы КЗ / Прозрачность
7. **Верификация** — split 50/50: объяснение слева + 3-шаговый процесс с green check справа
8. **Превью заявок** — 5 строк таблицы, нижние 2 размыты, overlay «89 заявок только для зарегистрированных»
9. **Финальный CTA** — navy секция, 2 панели (Заказчик / Исполнитель) с разными кнопками
10. **Footer** — 4 колонки: о компании + контакты + Платформа + Исполнителям + Заказчикам

## Применено из ui-ux-pro-max
- Паттерн: Marketplace / Directory (Hero → Categories → Trust → CTA)
- Стиль: Social Proof-Focused (stats перед CTA, verified badges, featured listings)
- Anti-pattern соблюдён: нет скрытых контактов (email/телефон в footer)
- Палитра: та же, что в ленте заявок (`#0F172A` navy, `#0369A1` blue, `#F8FAFC` bg)
- Типографика: Lexend 700 (hero, stats, section H2) + Source Sans 3 (body, nav, table)
